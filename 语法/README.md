- [slice切片(动态数组)](./9.2Slice.go)
长度不固定
    - 切片拥有**长度**和**容量**。
        - 长度是它所包含的元素个数。
        - 容量是从它的第一个元素开始数，到其底层数组元素末尾的个数。
```go
//slice 的底层是数组指针,数据结构如下
struct Slice
{   
    byte*    array;       // 实际数据
    uintgo    len;        // 长度
    uintgo    cap;        // 容量

};
```

- [接口](./14.1接口.go)
go里面没有类和继承的概念。
接口类型是由一组方法定义的集合。
接口类型的值可以存放实现这些方法的任何值
```go
type 接口类型名 interface{
    方法名1( 参数列表1 ) 返回值列表1
    方法名2( 参数列表2 ) 返回值列表2
    …
}
```
