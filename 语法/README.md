- [切片(动态数组)](./9.2Slice.go)
长度不固定
    - 切片拥有**长度**和**容量**。
        - 长度是它所包含的元素个数。
        - 容量是从它的第一个元素开始数，到其底层数组元素末尾的个数。
```go
//slice 的底层是数组指针,数据结构如下
struct Slice
{   
    byte*    array;       // 实际数据
    uintgo    len;        // 长度
    uintgo    cap;        // 容量

};
```
