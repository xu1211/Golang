- [接口](./14.1接口.go)
go里面没有类和继承的概念。
接口类型是由一组方法定义的集合。
接口类型的值可以存放实现这些方法的任何值
```go
type 接口类型名 interface{
    方法名1( 参数列表1 ) 返回值列表1
    方法名2( 参数列表2 ) 返回值列表2
    …
}
```
